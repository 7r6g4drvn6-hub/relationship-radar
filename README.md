# 关系雷达

一个本地网页小程序，把“朋友或恋人”的关系观察整合成五个模块：

- 雷达测试：从亲密、心动、双向、安全、未来、稳定六个维度记录双方感受。
- 默契卡：从题池抽取 10 题，对比两个人对关系、节奏、联系、关心方式等期待。
- 边界卡：从题池抽取 12 项，看哪些靠近方式对双方都舒服。
- 互动日记：记录互动类型、主动方、感受、温度、靠近感、压力感和下一步，影响稳定性画像。
- 剧情选择：从场景题池抽取 10 题，通过场景反应看关系信号。
- 关系画像：展示主信号、需要照顾的维度、关系节奏、画像可信度、六维条形分布和可聊的问题。
- 分享结果：两个人都标记完成后，可复制关系画像发给对方。
