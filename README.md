# 关系雷达

一个本地网页小程序，把“朋友或恋人”的关系观察整合成五个模块：

- 雷达测试：从亲密、心动、双向、安全、未来、稳定六个维度记录当前作答人的感受。
- 默契卡：从题池抽取 10 题，分别记录两个人对关系、节奏、联系、关心方式等期待。
- 边界卡：从题池抽取 12 项，分别记录当前作答人觉得舒服的靠近方式。
- 互动日记：记录互动类型、主动方、感受、温度、靠近感、压力感和下一步，影响稳定性画像。
- 剧情选择：从场景题池抽取 10 题，通过场景反应看关系信号。
- 关系画像：展示主信号、需要照顾的维度、关系节奏、画像可信度、六维条形分布和可聊的问题。
- 分享结果：每个人只做自己的题目，完成后复制带数据的链接发给对方；双方都完成后可复制双人画像。

## 打开方式

```bash
open /Users/xuxuan/Documents/codex/relationship-radar/index.html
```

数据保存在当前浏览器的本地存储里，不会上传。分享链接只携带作答人选择的题目答案、边界选择、剧情选择和名字。

## 发给别人

公开链接可直接发给对方。A 做完后复制链接给 B，B 打开后会自动带入 A 的结果并切到 B 作答；B 完成后也可以把链接发回给 A。更多说明见 `SHARE.md`。
